## 🌅 처음 시작하기

### 1. 비밀번호 만들기
- Google 로그인 후 **비밀번호 4자 이상**을 만드세요
- 이 비밀번호는 내 묵상·도트·결단을 안전하게 잠그는 열쇠예요
- **잃어버리면 다시 만들 수 없어요.** 꼭 기억하세요

### 2. 24단어 복구 코드
- 비밀번호를 잊었을 때 **유일한 회복 수단**이에요
- 종이에 적거나 비밀번호 매니저(1Password, Bitwarden 등)에 보관하세요
- 한 번 닫으면 다시 볼 수 없어요

### 3. 샘플 목표 (선택)
- 영적 성장을 위한 예시 4개를 미리 깔아둘 수 있어요
- 빈 상태로 시작해도 OK

## 🌙 하루 흐름

### 아침
1. **📖 오늘의 말씀** — 4파트 통독 본문이 자동으로 떠요
2. **✏️ 묵상 노트** — 떠오른 한 줄을 적어 보세요. 1초 뒤 자동 저장
3. **🙏 오늘의 결단** — `[+ 새 결단 적기]` → 한 줄 적기

### 낮
1. **결단 카드 ⋮⋮ 핸들**을 시간표로 끌어 옮겨 박기
2. **빈 시간 클릭** → 한 줄 입력으로 시계부 즉시 기록
3. **슬롯 클릭** → 평가 모달 (3초 룰: 1~4 키로 빠르게)

### 저녁 (18시 이후)
1. 사이드바 **🌙 저녁 통합 루프** (펄스로 신호)
2. 7단계가 한 페이지에 펼쳐짐 → 자유롭게 스크롤하며 진행
3. 토요일이면 8단계, 월말 토요일은 9단계로 자동 확장
4. **내일의 결단** 입력하면 다음 날 오늘 화면에 자동으로 떠요

## 💡 자주 쓰는 단축키

평가 모달에서:
- `1` `2` `3` `4` — 잘 했어요 / 조금 했어요 / 다른 걸 했어요 / 못 했어요
- `5` `6` `7` `8` `9` — 만족도 1~5
- `Enter` — 저장
- `Esc` — 닫기

저녁 루프에서:
- `← →` 화살표 — 이전 / 다음 섹션
