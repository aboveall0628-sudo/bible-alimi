# Sanctum OS — 이미지 자산 제작 프롬프트 v1

**작성**: 2026-05-20
**용도**: Claude (claude.ai) 새 대화에 통째로 복붙해서, 기존 자산 점검 + 새 톤 정합 + 안 맞으면 새로 만들기까지 한 묶음으로 진행하기 위한 명세서
**기조**: **"이미 있으면 보여줘라, 없으면 만들어라"**

---

## 사용 흐름 (단계별)

1. [claude.ai](https://claude.ai) 새 대화 열기
2. 다음 현재 자산 파일 첨부 (claude.ai 입력창 클립 아이콘):
   - `assets/og-image.png`
   - `assets/favicon-32.png`
   - `assets/favicon-16.png`
   - `assets/notification-icon.png`
   - `assets/notification-badge.png`
3. 아래 §4 코드 블록(노란 자리) 통째로 복붙해서 전송
4. 클로드가 자산별로:
   - 새 톤(올리브)에 이미 자리잡혀 있으면 → "OK 그대로" 알림
   - 안 맞으면 → 새 SVG 코드 줌
5. SVG 결과 받으면 §5 따라 PNG로 변환
6. `assets/` 폴더에 같은 파일명으로 덮어쓰기 → git push

## 5. SVG → PNG 변환 방법

클로드한테 SVG 받은 후 PNG로 만들기. 3가지 방법 중 편한 거 고르세요.

### 옵션 A — 브라우저 (가장 단순, 1200×630 OG 추천)

1. 메모장 같은 데에 받은 SVG 코드 붙여넣고 `temp.svg`로 저장
2. 더블클릭 → 브라우저로 열기
3. 우클릭 → "다른 이름으로 이미지 저장" → PNG 선택
4. ⚠️ 단점: 브라우저는 viewBox 사이즈 그대로 export. og-image(1200×630)는 OK, favicon(32×32)은 너무 작아서 별도 사이즈 조정 필요

### 옵션 B — ImageMagick (PowerShell, 정확한 사이즈)

설치 (한 번만): https://imagemagick.org/script/download.php#windows

```powershell
# OG 이미지
magick convert -background none -density 300 -resize 1200x630 input.svg og-image.png

# favicon 사이즈별
magick convert -background none -density 300 -resize 32x32 input.svg favicon-32.png
magick convert -background none -density 300 -resize 16x16 input.svg favicon-16.png

# notification 자산
magick convert -background none -density 300 -resize 192x192 input.svg notification-icon.png
magick convert -background none -density 300 -resize 96x96 input.svg notification-badge.png
```

### 옵션 C — Figma / 디자인 도구

SVG 코드 붙여넣고 "Export as PNG" → 사이즈 1x/2x 선택 → 다운로드

## 부록 — currentColor SVG는 손 안 댐

`sanctum-mark.svg`, `sanctum-wordmark-quiet.svg`, `notification-mark.svg`는 `stroke="currentColor"` 라서 CSS의 `color` 값을 자동으로 따라가요. 즉:

- 라이트 모드 → `--accent: #5F6F4A` 자동 적용 → 올리브
- 다크 모드 → `--accent: #8FA078` 자동 적용 → 밝은 올리브

이미 디자인 시스템 v1 토큰 갱신(2026-05-16)으로 자동 정합 완료. 추가 작업 X.
